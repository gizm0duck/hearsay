# Hearsay
Pub-sub library with active_record and action_controller hooks

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'hearsay'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install hearsay
```
